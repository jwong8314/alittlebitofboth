Tutorial¶

Setup:¶

Clone the latest tflon repository and ensure the following requirements: -pandas -tensorflow - Before using tflon, we use the following lines to import the necessary dependancies

import pandas as pd
import tensorflow as tf
import tflon

Example Dataset:¶

For the purposes of this tutorial, we will be using the Wisconsin Diagnostic Breast Cancer (WDBC) dataset found at https://archive.ics.uci.edu/ml/machine-learning-databases/breast-cancer-wisconsin/wdbc.data

The data has 30 real valued features including
  1. radius (mean of distances from center to points on the perimeter)
  2. texture (standard deviation of gray-scale values)
  3. perimeter
  4. area
  5. smoothness (local variation in radius lengths)
  6. compactness (perimeter^2 / area - 1.0)
  7. concavity (severity of concave portions of the contour)
  8. concave points (number of concave portions of the contour)
  9. symmetry
  10. fractal dimension (“coastline approximation” - 1)

We seek to construct a classifier to determine if the biometrics correspond to a malignant (212) or benign (357) using the 569 samples.

Pre-Processing:¶

The data is in the form of comma separated values(csv): ID,Malignant/Benign,Features. Using pandas, we extract ‘targ’ with a binary encoding 1 for malignant and 0 otherwise. And, ‘desc’ for the features.

    df = pd.read_csv("./wdbc.data" , header=None)
    df[1] = df[1].apply(transform) 
    df = df.iloc[:,1:]
    targ, desc = tflon.data.Table(df).split([1])

where transform is defined as:

def transform(x):
    if x == 'M':
        return 1
    else:
        return 0

Then, we create a tflon feed which essentially stores the ‘targ’ and ‘desc’ as a dictionary.

    feed = tflon.data.TableFeed({'desc':desc, 'targ':targ})

Defining a Model:¶

To define a new model, we create a new class.

class NeuralNet(tflon.model.Model):
    def _model(self):
        I = self.add_input('desc', shape=[None, 30])
        T = self.add_target('targ', shape=[None, 1])

        net = tflon.toolkit.WindowInput() |\
              tflon.toolkit.Dense(20, activation=tf.tanh) |\
              tflon.toolkit.Dense(5, activation=tf.tanh) |\
              tflon.toolkit.Dense(1)
        L = net(I)

        self.add_output( 'pred', tf.nn.sigmoid(L) )
        self.add_loss( 'xent', tflon.toolkit.xent_uniform_sum(T, L) )
        self.add_loss( 'l2', tflon.toolkit.l2_penalty(self.weights) )
        self.add_metric( 'auc', tflon.toolkit.auc(T, L) )

For this example, we have a dense layer of 5 neurons stacked on 20 neurons all with tanh activation function. Also, we use AUC to eveluate the model.

Train the Model:¶

To train the model, we instantiate the model:

    NN = NeuralNet()

Then, define training parameters, in our case 1000 iterations:

    trainer = tflon.train.OpenOptTrainer( iterations=100)

And, finally train it:

        NN.fit( feed, trainer, restarts=2 )

After each training we can assess the neural net with AUC:

        metrics = NN.evaluate(feed)
        print "AUC:", metrics['auc']

Putting it all together:¶

import pandas as pd
import tensorflow as tf
import tflon

class NeuralNet(tflon.model.Model):
    def _model(self):
        I = self.add_input('desc', shape=[None, 30])
        T = self.add_target('targ', shape=[None, 1])

        net = tflon.toolkit.WindowInput() |\
              tflon.toolkit.Dense(20, activation=tf.tanh) |\
              tflon.toolkit.Dense(5, activation=tf.tanh) |\
              tflon.toolkit.Dense(1)
        L = net(I)

        self.add_output( 'pred', tf.nn.sigmoid(L) )
        self.add_loss( 'xent', tflon.toolkit.xent_uniform_sum(T, L) )
        self.add_loss( 'l2', tflon.toolkit.l2_penalty(self.weights) )
        self.add_metric( 'auc', tflon.toolkit.auc(T, L) )
def transform(x):
    if x == 'M':
        return 1
    else:
        return 0
if __name__=='__main__':
		
    df = pd.read_csv("./wdbc.data" , header=None)
    df[1] = df[1].apply(transform) 
    df = df.iloc[:,1:]
    targ, desc = tflon.data.Table(df).split([1])
    feed = tflon.data.TableFeed({'desc':desc, 'targ':targ})
    NN = NeuralNet()
    
    trainer = tflon.train.OpenOptTrainer( iterations=100)
    
    with tf.Session():
        NN.fit( feed, trainer, restarts=2 )
        metrics = NN.evaluate(feed)
        print "AUC:", metrics['auc']
